# Checklist
Lumen API

## Setup
1. Clone this repo, `git clone https://github.com/budhilaw/checklist.git`
2. Migrate database, `php artisan migrate:fresh`
3. Seed the database, `php artisan db:seed`
4. Start, `php -S localhost:8000 -t public`
